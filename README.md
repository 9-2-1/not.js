# Not.js，共创世界次世代的 JSON 处理器。

## 优点

- 语法优雅到如同在 JS 操作 JSON。
- 支持 cache 系统，避免重复解析。
- 完全兼容其它插件。

## 编译

```bash
npm i
npm run build
```

## 使用

### 自定义扩展

请参照 [我的文章](https://www.ccw.site/post/109edca6-8fd7-4e9c-8462-dcc06ec38988)。

请对 `dist/main.js` 进行这个操作，然后即可在测试插件中使用。
